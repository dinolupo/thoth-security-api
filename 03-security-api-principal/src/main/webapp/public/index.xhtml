Can a Principal object be generated from HttpServletRequest? No roles, just a Principal with a given name?

Yes!

Below in the Principal/Role Information section where you can see the Principal name. Submit the form below to change the principal's name. After submitting the form, the page will refresh and you'll see the Principal name update. It is important to note the Principal name is actually coming from the Principal object! It's not just a JSF bean with a stored query parameter. That is the actual Java EE server Principal object.
