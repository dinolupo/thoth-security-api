When I get to the important code (the code I write to create the Principal and groups) can I use CDI to get an instance of some object which will help me get the principal and group information?

Yes!

Java EE Security API is a 1st class CDI citizen. So CDI can definately be used.

In this example, TestHttpAuthenticationMechanism uses...

@Inject
private IdentityStoreHandler identityStoreHandler;

...to inject an IdentityStoreHandler. The IdentityStoreHandler is a Java EE Security API class designed to help you get principal and role information.
