Can the Security API code get up and running embedded within a WAR? Does each request to this application run the Security API code?

Yes!!

The TestHttpAuthenticationMechanism class is created. On each request to this application, the TestHttpAuthenticationMechanism#validateRequest() method is executed. If you keep reloading this page, you can see results from the log file below that indeed the Security API code is being executed.
